Introduction

EasyUsb est une mini bibliothèque (actuellement écrit en c et c++) proposent une manipulation basique mais simple de périphérie usb. Elle s'appuyant sur la libusb1.0. il vous faudra donc, au préalable l'installer.

  • Auteur : Maleyrie Antoine
  • Début du projet : 26 Août 2011
  • Version actuel : 0.4
  • Site web : http://antoine163.github.io/easyusb/
  • Compilateur supporter : GCC
  • Licence MIT.

Installation

Commencer par télécharger le projet.

$ git clone https://github.com/antoine163/easyusb

Incluez dans votre projet, les fichiers ce trouvant dans le dossier de la versions (C ou C++) que vous souhaitez utiliser. Vous devez aussi installer la libusb1.0.

Utilisation

Un exemple et donner dans la documentation pour chaque versions (C ou C++). Vous devez aussi lier a vautre projet la libusb1.0 (-lusb-1.0).

Documentation

Les fichiers ont été conçus de sort à générai de la documentation automatique grâce au logicielle Doxygen. Vous pouvez générai la documentation en appellent doxygen avec son fichier de configuration se trouvent dans le dossier doc.
